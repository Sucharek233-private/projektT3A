                                             Jasně, ASCII layout je skvělý způsob, jak si rychle představit rozložení. Tady máš čistý, přehledný návrh, který odpovídá tomu, co jsme řešili — šibenice vlevo, slovo nahoře, klávesnice dole, info vpravo.

---

# ?? ASCII layout návrh pro hru Šibenice

```
+---------------------------------------------------------------+
|                         Š I B E N I C E                       |
+---------------------------------------------------------------+

+-------------------------+-------------------------------+
|                         |                               |
|        ________         |   Kategorie: Zvíře            |
|       |        |        |   Pokusy: 6 / 10              |
|       |        O        |   Špatná písmena: R, T, M     |
|       |       /|\       |                               |
|       |       / \       |   [Nápověda]   [Restart]      |
|       |                  |                               |
|     __|__                |                               |
|    /     \               |                               |
|   /       \              |                               |
+-------------------------+-------------------------------+

                        _  _  _  _  _  _ 
                      ( _  _  _  _  _  _ )

+---------------------------------------------------------------+
|  Q  W  E  R  T  Y  U  I  O  P                                |
|   A  S  D  F  G  H  J  K  L                                  |
|     Z  X  C  V  B  N  M                                      |
+---------------------------------------------------------------+
```

---

# ?? Co ten layout ukazuje

### **Levý panel**
- Velká šibenice s postavičkou.
- Místo pro animace nebo postupné dokreslování.

### **Horní střed**
- Název hry.

### **Horní střed pod názvem**
- Hádáné slovo jako `_ _ _ _ _`.

### **Pravý panel**
- Kategorie slova.
- Počet pokusů.
- Špatná písmena.
- Tlačítka jako Nápověda / Restart.

### **Spodní část**
- Virtuální klávesnice ve třech řadách.

---

Pokud chceš, můžu ti udělat i **alternativní layout** — třeba minimalistický, komiksový nebo úplně jiný styl. Nebo ti ho upravím podle toho, jak velké okno hry plánuješ.
